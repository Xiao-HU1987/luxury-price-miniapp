## 二、微信小程序发布流程

### 步骤 1：填写 AppID

打开 [project.config.json](file:///Users/huxiao/Public/测试项目-1-26.6.27/project.config.json)，将 `appid` 改为你的真实 AppID：

```json
{
  "appid": "wx1234567890abcdef",  // 替换为你的 AppID
  ...
}
```

---

### 步骤 2：切换生产环境

打开 [utils/config.js](file:///Users/huxiao/Public/测试项目-1-26.6.27/utils/config.js)，将 `ENV` 改为 `production`：

```javascript
const ENV = 'production';  // development → production

const CONFIG = {
  production: {
    BASE_URL: 'https://你的域名.com',  // 替换为你的服务器域名
    DEBUG: false
  },
  ...
};
```

---

### 步骤 3：配置服务器域名

登录微信公众平台 → 开发管理 → 开发设置 → 服务器域名：

**request 合法域名**：
```
https://你的域名.com
```

**注意事项**：
- 必须是 HTTPS
- 域名必须已备案
- 每月只能修改 5 次

---

### 步骤 4：真机调试

在微信开发者工具中：

1. 点击顶部「真机调试」按钮
2. 扫码在手机上打开小程序
3. 测试所有核心功能
4. 查看控制台是否有报错

> **重要**：开发者工具的模拟环境与真机有差异，必须真机测试！

---

### 步骤 5：上传代码

在微信开发者工具中：

1. 点击顶部「上传」按钮
2. 填写版本号（如：1.0.0）
3. 填写项目备注（如：首次发布，核心功能上线）
4. 点击「上传」

上传成功后，代码会进入微信后台的「开发版本」。

---

### 步骤 6：提交审核

登录微信公众平台 → 版本管理：

1. 点击「提交审核」
2. 填写审核信息：
   - 功能页面：填写核心页面路径
   - 测试账号：如需登录功能，提供测试账号
   - 补充说明：简要描述小程序功能
3. 提交等待审核

**审核时间**：通常 1-3 个工作日

---

### 步骤 7：发布上线

审核通过后：

1. 登录微信公众平台 → 版本管理
2. 点击「发布」
3. 选择发布方式：
   - **全量发布**：所有用户可见（推荐）
   - **分阶段发布**：按比例逐步开放（适合大流量）

发布成功后，用户在微信搜索小程序名称即可使用！

## 五、常见问题

**Q：审核被拒绝怎么办？**
A：查看拒绝原因，修改后重新提交。常见原因：类目不符、功能描述不清、缺少隐私协议。

**Q：服务器域名配置失败？**
A：检查：1）域名是否 HTTPS；2）SSL证书是否有效；3）域名是否备案。

**Q：微信登录失败？**
A：检查：1）AppID/AppSecret 是否正确；2）服务器域名是否配置；3）后端是否正常运行。

**Q：支付回调失败？**
A：检查：1）回调地址是否公网可访问；2）是否使用 HTTPS；3）商户证书是否正确。
